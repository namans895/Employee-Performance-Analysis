# Employee Performance Analysis

## Project Overview
This project analyzes employee performance data using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Exploratory Data Analysis
- Department-wise Analysis
- Promotion Eligibility Analysis
- Data Visualization

## Key Findings
- IT department achieved highest performance.
- Attendance positively impacted performance.
- Training hours improved employee productivity.
- 124 employees were eligible for promotion.
